Blog Landing Page
Overview
This project is a landing page design for a blogging site. It conveniently features all there is to offer on one page. Simply clicking on the navigation bar or the images will direct you to the appropriate page. The design was inspired by a friend who’s a bold outspoken writer, poet and podcast host.
